# Roadmap

## Limitações Atuais

- Ainda não tem frontend
- Ainda não tem API
- Ainda não salva histórico de análises
- Depende de CSVs em formato compatível
- Thresholds de insight ainda são simples
- Suporte inicial focado no MVP
- Mapeamento de colunas reais de Garage61 ainda não implementado
- Não há integração automática com Garage61

## Próximos Passos Sugeridos

### Curto Prazo

- Criar API FastAPI
- Criar endpoint de upload de CSV
- Melhorar mapeamento de colunas reais
- Adicionar persistência das análises
- Melhorar insights por curva/trecho

### Médio Prazo

- Conectar frontend React
- Implementar autenticação
- Adicionar visualização gráfica de telemetria
- Suportar múltiplos simuladores
- Adicionar análise de consistência entre voltas

### Longo Prazo

- Integração automática com Garage61
- Análise de múltiplas voltas do mesmo piloto
- Insights de estratégia de corrida
- Análise de traçado ideal
- Geração de áudio com insights
