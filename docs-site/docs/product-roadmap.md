# Product Roadmap

## Vision

AI Lap Simracing is an AI-powered simracing coach that helps drivers understand where they are losing time and what to train next.

## Product Phases

### MVP Local/Web
- Local CSV analysis
- Basic telemetry comparison
- Text report generation
- Simple web interface

### Reference Lap Collection
- Garage61 reference collector
- Reference candidate management
- Admin endpoints for curation
- Auto-fetch reference on demand

### Garage61 Integration
- Direct Garage61 API integration
- Automatic telemetry import
- Real-time analysis

### User Accounts
- Authentication system
- Personal dashboards
- Analysis history
- Progress tracking

### Analysis History
- Historical lap comparisons
- Trend analysis
- Performance metrics over time
- Improvement visualization

### Creator/Influencer Packs
- Pro driver reference laps
- Influencer-curated setups
- Premium coaching content
- Community features

### Paid Beta
- Beta access program
- Early adopter pricing
- Feedback collection
- Feature prioritization

### SaaS Subscription
- Tiered subscription plans
- Unlimited analysis
- Advanced features
- Priority support

## Monetization

### Initial Monetization Strategy
- **One-off analysis**: Single lap analysis purchases
- **Analysis packs**: Bulk analysis credits
- **Subscription**: Monthly/yearly plans for unlimited access
- **Influencer/reference lap packs**: Premium reference data from pro drivers
- **Future human coaching add-on**: AI + human coaching hybrid

## Technical PRs

### Next Steps
1. **Garage61 reference collector**
   - Build system to collect reference laps from Garage61
   - Store and categorize reference data

2. **Reference candidates/admin endpoints**
   - Admin interface for reference lap curation
   - Candidate review workflow
   - Quality control system

3. **Auto-fetch reference on demand**
   - Automatic reference lap retrieval
   - Smart matching based on car/track
   - Cache optimization

4. **CSV filename validation**
   - Input validation for CSV uploads
   - Error handling and user feedback
   - File format verification

5. **Coaching insights improvement**
   - Enhanced AI coaching algorithms
   - More specific actionable advice
   - Corner-by-corner analysis

6. **Deploy beta**
   - Production deployment
   - Beta user onboarding
   - Monitoring and analytics
